## What This Does
Control addressable LED strips from your phone or computer.  
Plug in your ESP32-S3, upload this code, and you've got a wireless lighting system with cool effects.

<br>

## What You Need
- ESP32-S3 board (CAN USE ANY) (the tiny one with USB-C)
- WS2812B LED strip (each LED can be a different color)
- Computer with Arduino IDE installed
- WiFi network your devices can connect to
